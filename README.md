# DevOps CI/CD Project 🚀

## Overview
Automated infrastructure provisioning and application deployment using Jenkins, Terraform, and Docker.

## Tech Stack
- Jenkins (CI/CD)
- Terraform (Infrastructure as Code)
- Docker (Containerization)
- AWS EC2

## Workflow
1. Code pushed to GitHub
2. Jenkins triggers pipeline
3. Terraform provisions EC2
4. Docker builds and deploys app

## Result
Fully automated deployment pipeline with infrastructure provisioning.
